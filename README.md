# Portfolio — Abderahmane Chaouche

Ce dépôt contient un portfolio one-page (HTML/CSS).

## À personnaliser (important)
- Remplace les images dans `assets/` :
  - `avatar.jpg`
  - `project1.jpg`, `project2.jpg`, `project3.jpg`
- Mets tes vrais liens dans `index.html` (LinkedIn, GitHub, liens de projets)
- Option : change le logo (AC) dans le header.

## Publier sur GitHub Pages (méthode simple)
1. Crée un repo GitHub (ex: `portfolio`)
2. Upload tous les fichiers (index.html, style.css, assets/)
3. Va dans **Settings → Pages**
4. **Source** : `Deploy from a branch`
5. **Branch** : `main` et dossier `/ (root)`
6. GitHub te donne l’URL publique.
